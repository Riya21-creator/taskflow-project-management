# TaskFlow - Project Management System

## Overview
TaskFlow is a web application where users can create projects, assign tasks, and track progress with role-based access.

## Features
- User Authentication (Login/Signup)
- Role-based access (Admin / Member)
- Project creation and team management
- Task creation and assignment
- Task status tracking (Todo, In Progress, Done)
- Dashboard with task statistics
- Overdue task tracking

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB
- Authentication: JWT

## How to Run Locally

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
